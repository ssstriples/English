## 48. 구두점 `d
- 주어 + 동사 + 목적어
- would, did, had
```
You`d(had) better run.

넌 / ~하는 게 낫겠다 / 뛰는걸
```
```
I`d(would) like to order spaghetti.

전 / 하고 싶습니다 / 주문을 / 스파게티로
```
```
Why`d(did) you do that?

왜 / 당신은 했습니까? / 그것을
```
---
```

어떻게 날 찾았니?

B: Your mather told me where you are.

너가 어딨는지 너의 어머니가 나에게 말했어.
```
---
```

우리 집에 가야겠어요.
```
```

뭘 보셨던 거예요.
```
```

저녁 식사하시고 싶은지 궁금해서요.
```
```

뭐라고 말씀하실 건가요? / 뭐라고 말씀하셨나요?
```

## 50. 감정형용사 ing / ed

- ed : 나의 감정이 ~하게 되어지다
```

```

## 51. on one`s way
- 평서문 / 의문문 + on one`s(the) way.
- ~하러 가는 길에
```

저는 / 가는 중입니다.
```
```
Are you on your way home?

당신은 / 집으로 가는 중인가요?
```
---
```

어디야?
```
